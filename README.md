# FamilyMap Privacy Policy Website

A simple React + Vite website hosting the FamilyMap privacy policy for Google Play submission.

## Features

- Clean, responsive privacy policy page
- COPPA compliant
- Hosted on GitHub Pages
- Auto-deployed with each push

## Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to view.

## Build & Deploy

```bash
npm run build
git add .
git commit -m "Update privacy policy"
git push
```

Changes will be live at: https://kasunldesilva.github.io/privacy-policy/

## Update Privacy Policy

Edit `src/App.jsx` and push changes to automatically redeploy.
